# TypeScript Node.js Project

A basic Node.js project setup with TypeScript support.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Development:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```

4. Run production build:
```bash
npm start
```

## Project Structure

- `/src` - TypeScript source files
- `/dist` - Compiled JavaScript files
